# Test Cases

시설물 자산 및 작업지시 관리자 화면 수동 테스트 케이스.

## 1. 테스트 환경

### 실행 방법

```bash
node server.js
```

접속:

```text
http://127.0.0.1:8080
```

### 테스트 계정

| ID | Password |
| --- | --- |
| `demo` | `demo1234` |

### 사전 조건

- 브라우저 localStorage 초기화 후 테스트 시작
- 필요 시 화면 상단 `초기화` 버튼으로 샘플 데이터 초기화
